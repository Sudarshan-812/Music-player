# 🎵  Music Player

A high-performance music player application built with React Native and Expo. Developed as part of the technical assignment for the Lokal team.

This application delivers a seamless audio experience with robust background playback, a custom dark-mode UI, and smooth infinite scrolling for browsing tracks.

## ✨ Key Features

* **🎧 Background Audio Playback:** Fully integrated with `react-native-track-player` to continue playing audio even when the app is minimized or the screen is locked.
* **📜 Infinite Scrolling:** Optimized list rendering for browsing large music libraries without performance drops.
* **🎨 Custom UI/UX:** Built with a sleek, custom dark theme (`#121212`) and vibrant accents (`#FF8216`).
* **⚙️ Native Android Integration:** Configured with Android foreground services, wake locks, and custom adaptive icons for a true native feel.
